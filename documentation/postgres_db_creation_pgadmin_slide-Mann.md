## Execute Database Creation Script in pgAdmin 4

- Open pgAdmin 4 and connect to your Render PostgreSQL server.
- In the left sidebar, expand your server and select the target database.
- Right-click the database and choose "Query Tool."
- In the Query Tool window, click the "Open File" icon (or use File > Open) and select your SQL script (e.g., assignment1.sql).
- Review the script to ensure it is correct.
- Click the "Execute/Run" button (lightning bolt icon) to run the script.
- Check the Messages tab for success or error messages.
- Your database tables and data should now be created as defined in the script.

---

Tip: You can also copy and paste SQL directly into the Query Tool if you prefer.
