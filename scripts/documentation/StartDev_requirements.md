# StartDev.ps1 Functional Requirements

**Reminder:** If any new permanent functionality is added to StartDev.ps1, update this requirements file to reflect the changes.

- Runs **'pnpm run dev'** from the project root in a new PowerShell window. **pnpm is required. Do not use npm or yarn.**
- After starting the server, opens the localhost URL to the correct port for dev in the default browser.
- Keeps the window open to show any errors.
- Outputs status messages for server start.
